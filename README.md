# Movie App

## Install

Install dependencies

```sh
$ yarn install
```

Duplicate `.env.sample` to `.env` and define properties.

### IOS specific

```sh
$ npx pod-install ios
```

## Running

```sh
$ npx react-native start
```

And in other terminal:

```sh
$ npx react-native run-ios --simulator="iPhone 5s"
```

```sh
$ npx react-native run-android
```
